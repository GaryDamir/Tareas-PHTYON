# Ejercicio 2: Practicando la Herencia en Python.

## Introduccion a la herencia.

### Clase de cuenta Bancaria y Mascota.

* *Cuenta bancaria*: Ejecutamos un programa el cual nos sirve como aplicacion de banco, con la cual podemos observar toda la informacion como el nombre de usuario, retirar dinero y hacer depositos.

* *Mascota*: Creamos un sistema para hacer la simulacion del cuidado de una mascota, mostrando informacion sobre su felicidad, entre otras.
